# Imágenes y diagramas

Esta carpeta contiene imágenes, fotos y diagramas utilizados en la documentación técnica del proyecto.

## Contenido

- Fotos del montaje de hardware
- Esquemas de cableado
- Diagramas de conexión
- Screenshots de configuración

## Formato recomendado

- **Fotos**: JPG/PNG
- **Diagramas**: PNG/SVG
- **Nombres**: Descriptivos en snake_case (ej: `rpi_mounting_01.jpg`)
